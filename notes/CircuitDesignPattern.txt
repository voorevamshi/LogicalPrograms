CircuitDesignPattern prevents from failure requests of a service when it is down.
It avaids burdon on the system and provide fallback failue event machanism.

class OrderService{

@Autowiredd

public void processOrder(Intger orderId){
	paymentService.getFoObject(orderId)
}

}
