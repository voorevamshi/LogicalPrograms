
Problem Statement
Build a Search API for Inventory search Management System
Backend Requirements
Create an API which will return the Inventory matching the filter criteria provided by the user. All the parameters to Search API are optional. Filter parameters will be part of AND conditions. Return values should the inventories that match the search criteria.

ID
Name
Category
Subcategory
Manufacturing Date
Expiry Date

price
stock 
Model
Seller
Location
……. Additional columns as you can assume

Required APIs
Search API to get all Inventory matching the filter

Expectations
Create a REST API
Open API standard YAML
Working code and demonstration
Use proper schema design
Proper validation of search parameters
Proper error handling
